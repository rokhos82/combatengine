Team definitions are JSON strings. There are a few required fields. Right now those are:

  • units - an array of unit objects (see below for required fields)

Unit definitions are also JSON strings. There are currently 3 required fields:

  • name
  • size
  • type

Here are some examples:

Red Fleet

      {{$ctrl.teams["Red"] | json}}

Blue Fleet

      {{$ctrl.teams["Blue"] | json}}
